Adicionar Emojis

Site: emojipedia.org

Para colocar basta copiar o código do emoji e inserir da forma: &#X[código do emoji]

Adicionar imagens

Basta utilizar a tag img, inserindo o nome do arquivo em src e a descrição da imagem em alt.

Pode-se usar um link em alt também

Formatação de textos

Negrito:

Usar a tag strong para deixar as palavras em Negrito

Itálico:

Usar a tag em para deixar as palavras em Itálico

Texto Marcado:

A tag mark cria um Marcador de Texto

Texto Pequeno:

A tag small deixa o Texto Pequeno

Texto Deletado:

A tag del faz a formatação de texto deletado

Texto Inserido:

A tag ins ou a tag u, deixa o texto sublinhado

Texto Sobrescrito

A tag sup torna possível escrever X20+3

Texto Subscrito

A tag sub que coloca caracter abaixo, como por exemplo em H20

Código-fonte

Tag code, usada para criar uma formatação monoespaçada

Citação:

Para fazer uma citação usa-se a tag q, e como dizia meu professor: Nem precisa usar aspas para a frase ficar assim.

Citação completa:

Para fazer uma citação em bloco se usa a tag blockquote e o parâmetro cite="" com o link, caso tenha, do local no qual foi retirado o texto.

Como por exemplo, essa citação de Nikola Tesla:

Deixem que o futuro diga a verdade e avalie cada um de acordo com o seu trabalho e realizações. O presente pertence a eles, mas o futuro pelo qual eu sempre trabalhei pertence a mim.

Abreviação:

Agora, para fazer abreviação, usamos a tag abbr preenchendo o parâmetro title="" com o significado da sigla, como por exemplo ao se falar de HTML basta passar o mouse por cima da sigla.
